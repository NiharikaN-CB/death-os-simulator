# 💀 Death OS Simulator

An interactive, chaotic OS-style web app built with Next.js.

## 🚀 Features

- 💀 Grim Reaper cursor tracking your mouse
- ⚡ Glitch screen effects
- ⏳ Countdown system
- 😂 Random troll popups
- 🪪 Survival Certificate generator (downloadable)

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- html2canvas

## 🧠 Concept
